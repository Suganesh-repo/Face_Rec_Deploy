Attendance Monitoring Using Face Recognition

This Project aims in creating a Web Application for Attendance Monitoring Using Face Recognition. To maintain Student Attendance Record effectively thereby reducing the time taken for staff especially when student strength is big. Admin can Add Student,Staff,Course and add photos with particular Register number, can make course Enrollment process. Students can enroll the courses, can mark their attendance after Staff opens the attendance process for the particular Course. It follows two factor Authentication. The attendance process takes place by entering the password and then face Recognition takes place. Staff can view daily Attendance Report, Overall Report, can mark OD.
